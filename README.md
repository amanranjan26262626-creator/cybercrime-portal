# Decentralized Cybercrime Reporting Portal

A production-ready Web3-enabled cybercrime reporting system with blockchain, AI chatbot, and comprehensive case management.

## Tech Stack

- **Frontend:** Next.js 14, TypeScript, Tailwind CSS, Ethers.js
- **Backend:** Node.js, Express, TypeScript
- **Blockchain:** Polygon, Hardhat, Solidity
- **Database:** PostgreSQL, MongoDB
- **Storage:** IPFS (Pinata)
- **AI:** Google Gemini API

## Project Structure

```
cybercrime-portal/
├── frontend/          # Next.js application
├── backend/           # Express API server
├── blockchain/        # Smart contracts
└── docs/             # Documentation
```

## Quick Start

### Prerequisites
- Node.js 20+
- Git
- API Keys: Pinata, Gemini, Polygon RPC

### Installation

1. Install dependencies:
   ```bash
   cd frontend && npm install
   cd ../backend && npm install
   cd ../blockchain && npm install
   ```

2. Setup environment variables (see `env.example.txt`)

3. Run development servers:
   ```bash
   # Frontend
   cd frontend && npm run dev
   
   # Backend
   cd backend && npm run dev
   ```
